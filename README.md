# Flutterを用いたスループットデータ収集用のアプリケーション
## 使い方説明
①保存するファイル名を入力する
②緑の測定開始ボタンを押す
③240秒経過すると測定が完了する
④データ保存ボタンを押す
⑤保存が完了すると画像２のようにファイルが保存されていることを確認できます
